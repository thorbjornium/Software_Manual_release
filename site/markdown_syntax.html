Escaping characters

You can use a backslash \ to escape the following characters:

CharacterName
\backslash
`backtick
*asterisk
_underscore
{ }curly braces
[ ]brackets
< >angle brackets
( )parentheses
#pound sign
+plus sign
-minus sign (hyphen)
.dot
!exclamation mark
|pipe

Escape backticks with two backticks.

``Use `code` in your Markdown file.``

Use four backticks to enclose code blocks.
