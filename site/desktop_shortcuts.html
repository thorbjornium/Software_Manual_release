Desktop Shortcuts

Keyboard shortcutAction
Ctrl + Esc or ⊞Open Start menu.
Ctrl + Shift + EscOpen Task Manager.
Ctrl + ShiftSwitch keyboard layout.
Alt + F4Close active window. (If no active window is present, a shutdown box appears.)
Ctrl + F5 or Ctrl + RRefresh current window.
Ctrl + Alt + TabView open apps.
Ctrl + ↑ ↓ ← → + SpaceSelect multiple items on desktop or File Explorer.
Alt + and an underlined letterRuns command for the underlined letter in apps.
Alt + TabSwitch between open apps while pressing Tab multiple times.
Alt + ←Go back.
Alt + →Go forward.
Alt + ▲Move up one screen.
Alt + ▼Move down one screen.
Alt + EscCycle through open windows.
Alt + SpaceOpen context menu for the active window.
Alt + F8Reveals typed password in Sign-in screen.
Shift + Click app buttonOpen another instance of an app from the Taskbar.
Ctrl + Shift + Click app buttonRun app as administrator from the Taskbar.
Shift + Right-click app buttonShow window menu for the app from the Taskbar.
Ctrl + Click a grouped app buttonCycle through windows in the group from the Taskbar.
Shift + Right-click grouped app buttonShow window menu for the group from the Taskbar.
Ctrl + ←Move the cursor to the beginning of the previous word.
Ctrl + →Move the cursor to the beginning of the next word.
Ctrl + ↑Move the cursor to the beginning of the previous paragraph
Ctrl + ↓Move the cursor to the beginning of the next paragraph.
Ctrl + Shift + ↑ ↓ ← →Select block of text.
Ctrl + SpaceEnable or disable Chinese IME.
Shift + F10Open context menu for selected item.
F10Enable app menu bar.
Shift + ↑ ↓ ← →Select multiple items.
⊞ + Number 0-9Open the app in number position from the Taskbar.
⊞ + TCycle through apps in the Taskbar.
⊞ + Alt + Number 0-9Open Jump List of the app in number position from the Taskbar.
⊞ + XOpen Quick Link menu.
⊞ + DDisplay and hide the desktop.
⊞ + MMinimize all windows.
⊞ + Shift + MRestore minimized windows on the desktop.
⊞ + HomeMinimize or maximize all but the active desktop window.
⊞ + Shift + ↑ Stretch desktop window to the top and bottom of the screen.
⊞ + Shift + ↓ Maximize or minimize active desktop windows vertically while maintaining width.
⊞ + Shift + ←Move active window to monitor on the left.
⊞ + Shift + →Move active window to monitor on the right.
⊞ + ← Snap app or window left.
⊞ + →Snap app or window right.
⊞ + S or QOpen Search.
⊞ + Alt + DOpen date and time in the Taskbar.
⊞ + TabOpen Task View.
⊞ + Ctrl + DCreate new virtual desktop.
⊞ + Ctrl + F4Close active virtual desktop.
⊞ + Ctrl + →Switch to the virtual desktop on the right.
⊞ + Ctrl + ←Switch to the virtual desktop on the left.
⊞ + POpen Project settings.
⊞ + AOpen Action center.
⊞ + IOpen Settings app.
BackspaceReturn to the Settings app home page.
